# 構造材料特性データベース / Structural Materials Properties Database

構造解析で参照できる材料特性データを閲覧するための静的 Web アプリケーションです。
GitHub Pages でホストされており、ブラウザからすぐに利用できます。

## 収録データ

303 件の材料を 11 カテゴリに分類して収録しています。

| カテゴリ | 件数 |
|---|---|
| 高分子 (Polymer) | 72 |
| エラストマー (Elastomer) | 66 |
| 木質 (Wood) | 50 |
| 複合材料 (Composite) | 19 |
| 接着剤 (Adhesive) | 18 |
| セラミックス (Ceramics) | 18 |
| フォーム (Foam) | 17 |
| 金属 (Metal) | 15 |
| ガラス (Glass) | 14 |
| コンクリート・グラウト (Concrete and Grout) | 12 |
| 地盤・土質 (Soil) | 2 |

各材料について、以下の情報を含みます。

- 線形弾性定数 (ヤング率、ポアソン比、密度など)
- 強度データ (降伏強度、引張強度、圧縮強度など)
- 非線形構成則パラメータ (J2 塑性、超弾性など)
- 直交異方性弾性定数 (木質材料・複合材料)
- ソルバー別パラメータマッピング (ANSYS, Abaqus, DOLFINx, LS-DYNA)

## 機能

### 閲覧・検索
- **大分類・中分類フィルタ** — サイドバーでカテゴリを選択して絞り込み
- **テキスト検索** — 材料名・ID・カテゴリ・出典などを横断検索
- **ソルバーフィルタ** — ANSYS / Abaqus / DOLFINx / LS-DYNA のマッピングを持つ材料に絞り込み
- **数値範囲フィルタ** — ヤング率・密度・降伏強度を上下限で絞り込み (SI / US 両対応)
- **並び替え** — 名称順、分類順、ヤング率・密度・降伏強度の降順、最近閲覧順
- **グリッド / テーブル表示切替** — カード表示と一覧テーブルを切替
- **詳細表示** — カードをクリックすると全特性・ソルバーマッピング・出典をリンク付きで表示

### 分析・可視化
- **Ashby 散布図** — 任意の 2 物性を軸に選択し、カテゴリ別に色分けして表示 (対数スケール対応)
- **ヒストグラム** — 選択した物性の分布をカテゴリ別に表示
- **統計ダッシュボード** — カテゴリ別の材料数・平均値・範囲を一覧表示
- **レーダーチャート** — 材料比較時に主要物性を多角形チャートで可視化
- **板厚別物性チャート** — 板厚依存の強度データをライン表示
- **類似材料検索** — 物性空間での距離に基づき類似材料を推薦

### ソルバー連携
- **ソルバーカード生成** — ANSYS / Abaqus / DOLFINx / LS-DYNA 向けの材料カードをワンクリックでコピー
- **直交異方性対応** — 木質・複合材料の直交異方性弾性定数を各ソルバーの形式で出力
- **部分データの安全な処理** — 不足パラメータがある材料では 0 埋めせず、既知定数のみ出力し不足分を警告コメントで表示

### エクスポート
- **CSV ダウンロード** — 現在のフィルタ結果を CSV ファイルとしてエクスポート
- **JSON ダウンロード** — フィルタ結果を JSON 形式でエクスポート

### 比較・共有
- **材料比較** — 最大 4 件を選択し、主要物性を横並びで比較
- **お気に入り** — ブラウザの localStorage に保存、コレクションとして絞り込み可能
- **最近見た材料** — 閲覧履歴を自動記録
- **URL 共有** — 検索条件・フィルタ・選択材料の状態を URL クエリパラメータに反映し、共有やブックマークが可能

### 表示
- **SI / US 単位系切替** — ヘッダーのボタンで Pa ↔ psi、kg/m³ ↔ lb/ft³ を切替
- **日本語 / 英語切替** — ヘッダーのボタンでカテゴリ名・UI ラベルを切替
- **ダークモード / ライトモード** — OS 設定を自動検出、手動切替も可能
- **レスポンシブデザイン** — デスクトップ・モバイル対応

## 技術構成

```
index.html          メインページ
css/style.css       スタイルシート
js/app.js           アプリケーションロジック
js/features.js      物性定義・ソルバーカード生成・フィルタ補助
js/charts.js        SVG チャート描画 (散布図・レーダー・ヒストグラム・折れ線・棒)
materials_db.json   材料特性データ (JSON)
```

- 外部ライブラリ・CDN への依存なし (HTML / CSS / JavaScript のみ)
- `fetch` による JSON 読み込み (GitHub Pages で動作)
- URL クエリパラメータによるビュー状態の永続化
- localStorage によるお気に入り・閲覧履歴の保持

## ローカルでの確認方法

```bash
# リポジトリをクローン
git clone https://github.com/Takayuki-Minagawa/materials-db.git
cd materials-db

# ローカルサーバーを起動 (Python 3)
python3 -m http.server 8080

# ブラウザで http://localhost:8080 を開く
```

`file://` プロトコルでは `fetch` が動作しないため、HTTP サーバー経由でアクセスしてください。

## データの利用に関する注意

- 本データベースは構造解析の**初期検討や学習を目的とした参考値の集約**であり、認証済みの試験データやミルシートの代替ではありません。
- 各材料の特性値は、製造元のデータシート、技術文献、および公的規格から取得しています。出典情報は各材料の詳細画面および JSON データ内に記載しています。
- 大ひずみ塑性、損傷、クリープ、疲労などの高度な解析には、実測による構成則データの使用を推奨します。
- 金属材料の非線形データの多くは、保守的な弾性完全塑性 (elastic-perfectly-plastic) モデルに基づいています。
- 木質材料、土質材料、接着剤の一部パラメータには、文献から推定した参考値を含みます。
- 本データベースの利用によって生じた損害について、作成者は一切の責任を負いません。
